Generationin computer terms, is the major changes on the technology the computer is/was being used. This may also refer to the release of computers or components because of a new hardware or technology. As a result of these advancements, computers now have wide varieties of applications. Now let us get down to how computers have developed until now.

 

A Commodore SuperPET SP9000

 

 

 

The 5 Main Generations of Computers

 

 

 

First Generation (1943-1958)

Vacuum Tube based.

 

Second Generation (1959-1964)

Transistor based.

 

Third Generation (1965-1970)

Integrated Circuit based.

 

Fourth Generation (1971-1989)

VLSI microprocessor based.

 

Fifth Generation (1990-Onwards)

ULSI microprocessor based.

 

 

 

1st Generation (1943-1958)

 

 

 

Cpl. Irwin Goldstein (foreground) sets the switches on one of the ENIAC's function tables at the Moore School of Electrical Engineering. (U.S. Army photo).
By Unidentified U.S. Army photographer, Public domain, via Wikimedia Commons

 

 

 

The first generation of computers used vacuum tubes as basic component for circuitry and memory for its CPU (Central Processing Unit). Vacuum tubes are large in size which made the first-generation computers take up a lot of space.

 

These tubes produce a lot of heatakin to lightbulbsand therefore it fuses often. It is also quite expensive, that only large organizations can afford it.

 

 

 

Some computers of this generation were:

 

 

ENIAC

EDVAC

UNIVAC

IBM-701

IBM-650

 

 

 

2nd Generation (1959-1964)

 

 

 

U.S. Census Bureau employees tabulate data using one of the agencys UNIVAC computers, ca. 1960.
By U.S. Census Bureau employees, Public domain, via Wikimedia Commons

 

 

 

The second generation of computers used transistors. Transistors are cheaper, compact, faster and consumed less power than the first-generation vacuum tubes. It also used magnetic cores as primary memory, as well as, magnetic tape and magnetic disks as secondary storage.

 

Also, in this generation, assembly language and high-level language like FORTRAN (Formula Translation) and COBOL (Common Business-Oriented Language) are used.

 

 

 

Some computers of this generation were:

 

 

IBM 1620

IBM 7094

CDC 1604

CDC 3600

UNIVAC 1108

 

 

 

3rd Generation (1965-1970)

 

 

 

.

A Honeywell Level 66/60 mainframe computer with its cabinet door open.

By Dorsetonian, CC BY-SA 4.0, via Wikimedia Commons

 

 

 

The third generation used Integrated Circuits (IC's) instead of transistors. A single IC contains many transistors, resistors and capacitors along with the associated circuitry. Using this helped reduce the size of computers and made them faster, compared to the second-generation computers.

 

 

 

Some computers of this generation are:

 

 

IBM-360 series

Honeywell-6000 series

PDP (Personal Data Processor)

IBM-370/168

TDC-316

 

 

 

4th Generation (1971-1989)

 

 

 

A processor Intel C4004 with grey traces. By Thomas Nguyen, CC BY-SA 4.0, via Wikimedia Commons

 

 

 

The fourth generation used the Very Large Scale Integrated (VLSI) circuits. Computers of this generation took advantage of microprocessor (or what we commonly know now as CPU), which made it much more powerful, compact, and affordable. This gave rise to the Personal Computer (PC) revolution.

 

 

 

Some computers of this generation are:

 

 

DEC 10

STAR 1000

PDP 11

CRAY-1 (Super Computer)

CRAY-X-MP (Super Computer)

 

 

 

5th Generation (1990-Onwards)

 

 

 

 

 

 

In the fifth generation, the VLSI technology has advanced to ULSI (Ultra Large-Scale Integration). This resulted in making of microprocessor chips having ten million electronic components.

 

This generation is based on parallel processing hardware and AI (Artificial Intelligence) software. All the higher-level languages like C and C++, Java, .Net etc. are used in this generation.

 

 

 

Some computers of this generation are:

 

 

Desktop

Laptop

NoteBook

UltraBook

ChromeBook
