Both hardware and software are both import components of a computer. Hardware has all the components to make it possible to make the device work, and the software, on the other hand, is the what gives instructions as to how the computer would function.

It is essential to learn the difference about the hardware and software, so that we can discern which is which in the event the device function erratically. And now, let us continue to learn more about the computer hardware and software.

 

 

 

Hardware

 

 

 

 

What is Hardware?

 

 

 

The computer hardware is the physical parts of a computer. It has all the components to make the device work. Without the hardware, the software would be useless since it wont be accessible.

 

 

3 Fundamental Elements of a Computer

 

 

1.                 System Unit

 

2.                 Input Devices

 

3.                 Output Devices

 

 

 

 

System unit

 

 

 

 

 

 

The system unit is the core of the computer. It contains the main components that is responsible for: processing and storing data; and controls all computer functions. This contains the motherboard, CPU, RAM, etc. Even including the case that houses these components.

 

 

 

Basic Parts of the System Unit

 

 

 

Motherboard

 

 

 

 

 

 

The motherboard is the most important part of the computer. It connects all of the parts of the computer together, and is the largest component on most computers.

 

 

 

 

CPU

 

 

 

 

 

 

Central Processing Unit (CPU) also called the processor, is the part of a computer that interprets and carries out instructions of a program by performing the basic arithmetic, logical, control and input/output (I/O) operations specified by the instructions.

 

 

 

Memory

 

 

 

Computer memory can be classified as either internal or external memory.

 

 

 

Internal Memory

 

 

 

Internal memory also called as the primary memory. It stores small amounts of data that can be accessed almost instantly as the computer is running.

 

 

 

2 Types of Internal Memory

 

 

 

RAM

ROM

 

 

RAM

RAM stands for Random Access Memory. It temporarily stores data as the CPU executes tasks. Having bigger RAM allows the computer to run faster, since the CPU has less data to read from the external memory (storage device).

 

ROM

ROM stands for Read-Only-Memory. It is non-volatile memory unlike RAM, which means it can retain data even when turned off. It is mainly used to start/boot up a computer.

 

 

 

External Memory

 

 

 

 

 

 

External memory also called as the secondary memory. It is a storage device that can store files permanently. It is not accessed by the CPU directly, but rather it was first attached to the main memory and then the CPU can access it.

 

 

 

 

Input Devices

 

 

 

 

 

 

Input devices are peripheral devices or accessories that are used as means to generate input for the computer. It lets the user to enter information into the computer.

 

 

 

Examples of Input Devices

 

 

Keyboard

Mouse

Scanner

Microphone

Camera/Webcam

 

 

 

 

Output Devices

 

 

 

 

 

 

Output devices are peripheral devices that can bring forth an output. After the computer completes a computation, it then integrates with the output device to generate the output. The output device converts data into a human readable formatlike music, graphics, text, audio, and/or video.

 

 

Examples of Output Devices

 

 

Monitor

Printer

Speaker/s

Headphones

Projector

 

 

 

 

SOFTWARE

 

 

 

 

 

 

 

What is Software?

 

 

 

Software is a set of instructions, programs or data that instruct a computer what to do. It usually refers to applications used to execute a specific task.

 

Software often falls into two categories: Application Software and System Software.

 

 

 

Application Software

 

 

 

Application softwareor apps, as we usually call it nowis what we mostly engage ourselves with when we use computers. These types of software are which help us perform tasks. It is also what makes computer popular and easy to use.

 

 

 

Examples of Application Software:

 

 

Microsoft Office Suite

Zoom

Google Chrome

PotPlayer

Discord

 

 

 

System Software

 

 

 

System software is a collection of software that is essential in managing the computer system. It is designed to operate, control and expand the processing power of a computer. System software also makes it easier for hardware and applications to interact and work together.

 

 

 

Examples of System Software:

 

 

Operating Systemssuch as Microsoft Windows, Ubuntu, Linux, etc.

Utility Softwaresuch as Anti-Virus, Network Managers, etc.

Device Driverssuch as USB drivers, Mouse Drivers, etc.

Firmwaresuch as BIOS, Embedded Systems, etc.
